# AsaDB Community Code of Conduct

The AsaDB community is a technical space for learning and building together.
We expect participants to be respectful, patient, specific, and honest about
what they know and what remains uncertain.

## Expected Behavior

- Discuss code and ideas without attacking people.
- Welcome beginners and explain review feedback constructively.
- Credit prior work, contributors, bug reporters, and external dependencies.
- Share reproducible evidence for correctness and performance claims.
- Protect private information and disclose security issues responsibly.
- Accept maintainers' decisions about project scope while remaining free to
  disagree respectfully.

## Unacceptable Behavior

Harassment, discrimination, threats, doxxing, deliberate disruption,
plagiarism, impersonation, and repeated bad-faith behavior are not accepted.

## Enforcement

Report conduct concerns privately to the project maintainer through the contact
method listed on the GitHub organization profile. Maintainers may edit or remove
content, restrict participation, or ban participants when necessary. Reports
will be handled as privately and fairly as practical.
